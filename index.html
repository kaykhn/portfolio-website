Project: Automated Program Verifier (Static Analysis)
Implemented a static program analyzer using abstract interpretation to verify properties (non-negativity, range bounds, delay invariants) on Java programs without executing them. I used Soot for Java bytecode parsing and pointer analysis, and Apron for tracking numerical variable ranges across control flow.

P.03
Project: Implementing Dynamic Declarative Power Sequencing
Implemented a Rust-based monitoring system for power networks in modern servers. The solution parses a user-defined specification of the hardware layout and generates a live dependency model that can issue I2C commands to hardware components and track abstract power states across the system. Layed out the foundational component of a larger power sequencing framework, enabling future automation.
RustI2CEmbedded Systems
